摩托车里程碑问题

李刚骑摩托车在公路上高速匀速行驶：

• 早晨7:00时看到里程碑上的数是一个两位数，它的数字之和是9

• 8:00时看里程碑上的两位数与7:00时看到的个位数和十位数颠倒了

• 9:00时看到里程碑上的数是7:00时看到的数的8倍

请问：李刚在7:00时看到的数字是多少?

问题分析

首先，我们需要理解题目给出的信息：

1. 设7:00时看到的两位数为10a + b，其中a是十位数字，b是个位数字

2. 根据题意，数字之和：a + b = 9

3. 8:00时看到的数字是10b + a（数字颠倒）

4. 9:00时看到的数字是8(10a + b)

5. 李刚匀速行驶，所以8:00到9:00行驶的路程等于7:00到8:00行驶的路程

由于李刚匀速行驶，相邻时间段行驶的路程相等：

8:00里程碑数 - 7:00里程碑数 = 9:00里程碑数 - 8:00里程碑数

即：

(10b + a) - (10a + b) = 8(10a + b) - (10b + a)

建立方程

根据题意，我们可以建立以下方程：

a + b = 9

(10b + a) - (10a + b) = 8(10a + b) - (10b + a)

首先简化第一个方程：

第一步：简化数字颠倒的差

(10b + a) - (10a + b) = 10b + a - 10a - b = 9b - 9a = 9(b - a)

第二步：简化第二个差

8(10a + b) - (10b + a) = 80a + 8b - 10b - a = 79a - 2b

第三步：建立等式

9(b - a) = 79a - 2b

解方程

现在我们有两个方程：

a + b = 9

9(b - a) = 79a - 2b

第一步：展开第二个方程

9(b - a) = 79a - 2b

9b - 9a = 79a - 2b

9b + 2b = 79a + 9a

11b = 88a

b = 8a

第二步：代入第一个方程

a + b = 9

a + 8a = 9

9a = 9

a = 1

第三步：求b的值

b = 8a = 8 × 1 = 8

第四步：确定7:00看到的数字

7:00看到的数字是 10a + b = 10 × 1 + 8 = 18

验证答案

我们得到的答案是：李刚在7:00时看到的数字是18

让我们验证一下：

• 7:00：数字18，数字之和 1 + 8 = 9 ✓

• 8:00：数字颠倒为81 ✓

• 9:00：数字是7:00数字的8倍，18 × 8 = 144 ✓

• 检查匀速行驶：

7:00到8:00行驶距离：81 - 18 = 63公里

8:00到9:00行驶距离：144 - 81 = 63公里 ✓

最终答案

李刚在7:00时看到的数字是 18

课堂练习

小明骑自行车匀速行驶：

• 上午10:00看到里程碑上的数是一个两位数，数字之和是7

• 11:00时看到的数字是10:00时数字的十位和个位颠倒

• 12:00时看到的数字是10:00时数字的6倍

请问：小明在10:00时看到的数字是多少？

提示1：设未知数

设10:00时看到的两位数为10x + y，其中x是十位数字，y是个位数字

提示2：根据题意建立方程

1. x + y = 7

2. 11:00看到的数字是10y + x

3. 12:00看到的数字是6(10x + y)

4. 匀速行驶：(10y + x) - (10x + y) = 6(10x + y) - (10y + x)

第一步：建立方程

x + y = 7

(10y + x) - (10x + y) = 6(10x + y) - (10y + x)

第二步：简化方程

9y - 9x = 60x + 6y - 10y - x

9y - 9x = 59x - 4y

9y + 4y = 59x + 9x

13y = 68x

y = (68/13)x

第三步：代入第一个方程

x + (68/13)x = 7

(13/13)x + (68/13)x = 7

(81/13)x = 7

x = 7 × 13 ÷ 81 = 91/81 ≈ 1.123

这不是整数，说明题目可能有误或者需要调整思路。

第四步：重新检查题目

实际上，这类问题通常有整数解。让我们尝试另一种思路：

设10:00数字为N，则11:00数字为数字颠倒，12:00数字为6N

由于匀速：11:00数字 - N = 6N - 11:00数字

2 × 11:00数字 = 7N

11:00数字 = 3.5N

但11:00数字是整数，所以N必须是偶数。

尝试N=16：数字和1+6=7 ✓，颠倒后61，61-16=45，6×16=96，96-61=35 ✗

尝试N=25：数字和2+5=7 ✓，颠倒后52，52-25=27，6×25=150，150-52=98 ✗

尝试N=34：数字和3+4=7 ✓，颠倒后43，43-34=9，6×34=204，204-43=161 ✗

实际上，这个特定问题可能没有整数解，或者题目数据需要调整。

结论

这个练习题目可能设计有误，但解题思路与原始问题相同。在实际教学中，教师可以调整数据确保问题有解。

总结

解题关键点

1. 正确设未知数：用10a+b表示两位数

2. 理解数字颠倒的含义：10b+a

3. 利用匀速行驶建立方程：相邻时间段行驶距离相等

4. 解方程组，注意a和b必须是0-9的整数

5. 验证答案是否符合所有条件

类似问题的通用解法

对于这类"里程碑问题"，可以总结出通用解法：

设初始数字为10a+b，数字和为s，倍数为k

则：

a + b = s

(10b + a) - (10a + b) = k(10a + b) - (10b + a)

化简后解方程组即可。
