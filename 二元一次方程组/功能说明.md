# 课件导航系统功能说明

## 📋 功能总览

课件导航系统现在包含三大核心功能，每个功能都可以通过`navigation.js`顶部的开关独立控制：

```javascript
const NAVIGATION_ENABLED = true;    // 导航栏开关
const TIMER_ENABLED = true;          // 倒计时工具开关
const SECTION_NAV_ENABLED = true;    // Section左右导航按键开关
```

---

## 1️⃣ 导航栏功能 (NAVIGATION_ENABLED)

### 功能描述
- 右上角固定的导航按钮（☰）
- 点击打开侧边栏，显示所有课件页面
- 快速跳转到任意课件页面

### 操作方式
- 点击右上角 **☰** 按钮打开导航栏
- 点击任意课件项跳转
- 按 **ESC** 键或点击遮罩层关闭导航栏

### 样式特点
- 蓝色渐变背景
- 滑入滑出动画
- 当前页面高亮显示

---

## 2️⃣ 倒计时工具 (TIMER_ENABLED)

### 功能描述
升级版倒计时工具，支持一键开始和重置

### 操作方式
1. **首次点击** → 开始计时（绿色变橙色，带脉冲动画）
2. **再次点击** → 重置计时并返回未计时状态（橙色变绿色）

### 显示位置
- 桌面端：右上角（导航按钮左侧）
- 移动端：右上角导航按钮下方

### 状态说明
- 🟢 **绿色** = 未开始（not-started）
- 🟠 **橙色** = 计时中（running，带脉冲动画）

### 视觉特点
- 渐变背景设计
- 悬停放大效果
- 计时时有脉冲动画
- 使用等宽字体显示时间

---

## 3️⃣ Section导航按键 (SECTION_NAV_ENABLED)

### 功能描述
在页面左右两侧添加导航按键，支持：
- ✅ Section内滚动导航
- ✅ 跨页面自动跳转
- ✅ 键盘快捷键支持

### 操作方式

#### 点击操作
- **左按钮（‹）** → 跳转到上一个section，如果是第一个section则跳转到上一个课件
- **右按钮（›）** → 跳转到下一个section，如果是最后一个section则跳转到下一个课件

#### 键盘快捷键
- **← 左箭头键** → 上一个section/上一页
- **→ 右箭头键** → 下一个section/下一页

### 智能特性

1. **平滑滚动**：Section间切换使用平滑滚动动画
2. **跨页跳转**：最后一个section按右键自动跳转到下一个课件
3. **禁用状态**：
   - 第一页第一个section时，左按钮禁用（半透明显示）
   - 最后一页最后一个section时，右按钮禁用（半透明显示）
4. **输入框保护**：在输入框（input/textarea）中不响应键盘导航

### 显示位置
- **左按钮**：屏幕左侧中央，垂直居中
- **右按钮**：屏幕右侧中央，垂直居中

### 视觉特点
- 紫色渐变半透明背景
- 毛玻璃效果（backdrop-filter）
- 悬停时外推动画
- 禁用时变灰且不可点击

---

## 🎨 统一设计语言

所有功能都采用统一的视觉设计：
- 🎨 **配色方案**：紫色渐变主题（#667eea → #764ba2）
- ✨ **动画效果**：流畅的缓动函数和3D变换
- 💫 **交互反馈**：悬停放大、阴影增强
- 📱 **响应式设计**：完美适配桌面端和移动端

---

## 📱 移动端适配

### 屏幕 ≤ 768px
- 导航栏全屏显示
- 导航按钮缩小到50x50px
- 倒计时器移至导航按钮下方
- Section按键缩小到45x60px

### 屏幕 ≤ 480px
- 倒计时器进一步缩小
- Section按键缩小到35x50px
- 字体大小相应调整

## 🔧 开发者配置

### 开关配置（navigation.js 顶部）
```javascript
// 修改这些值为 false 可以禁用对应功能
const NAVIGATION_ENABLED = true;    // 导航栏
const TIMER_ENABLED = true;          // 倒计时
const SECTION_NAV_ENABLED = true;    // Section导航按键
```

### Section标记要求
要使用Section导航功能，HTML需要正确标记section：
```html
<section data-section="1" id="section1">
    <!-- 内容 -->
</section>
<section data-section="2" id="section2">
    <!-- 内容 -->
</section>
```

### 页面顺序配置
页面跳转顺序由`navigation.js`中的`pages`数组定义：
```javascript
const pages = [
    { url: '0.封面页.html', title: '封面页', icon: '📄', desc: '...' },
    { url: '1.目录页.html', title: '目录页', icon: '📚', desc: '...' },
    // ... 其他页面
];
```

## 🎯 使用建议

1. **课堂演示**：开启所有功能，方便快速导航和计时
2. **学生练习**：可以只开启倒计时和Section导航，隐藏导航栏
3. **考试模式**：可以关闭所有导航功能，只保留基础内容

---

## 🐛 已知问题和注意事项

1. **Section检测**：Section导航依赖于`data-section`属性，请确保所有需要导航的section都正确标记
2. **键盘快捷键**：在输入框中输入时，左右箭头键会被正确忽略
3. **移动端操作**：触摸设备上使用点击操作，键盘导航在移动端不可用
